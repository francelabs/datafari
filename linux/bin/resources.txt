Where to change the amount of RAM for the different services :

You can check the default JVM RAM configuration of Datafari CE in monoserver_community_memory.properties , and in the equivalent file of the Datafari EE version main_enterprise_memory.properties.
This file is located in : $DATAFARI_HOME/bin/deployUtils/$YOUR_VERSION_memory.properties

For more information see : https://datafari.atlassian.net/wiki/spaces/DATAFARI/pages/495353895/Adjust+components+RAM+consumption
